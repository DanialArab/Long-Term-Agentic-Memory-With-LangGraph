# Long-Term Agentic Memory With LangGraph

- Mental framework to add memory to agents -> **LangMem** that supports vector DB that provides searchable, shareable, persistent storage than can ve updated immediately by the agent or in the background by a helper agent 
-  What info we need to store in long term memory and also when and what to retrieve when needed
-  we need to also decide when to update the stored information in each agent loop iteration or in the background over time, these are questions we need to have clarity on 

## Types of memories

- Semantic memories: like facts like date of birth
- Episodic memories: experiences that can help an agent remember how to do tasks
- Procedural memories: rules for an agent to follow
